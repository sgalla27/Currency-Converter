# Currency Converter Web App

This web-based currency converter allows users to quickly convert between different world currencies.  
Built using **Vue.js** and **HTML/CSS**, with a focus on a clean, user-friendly interface.  



## Features
- Real-time currency conversion with updated exchange rates from external API
- Intuitive UI with responsive layout  
- Custom icons and styled buttons for improved usability
